# Quiz Trivia Game
A simple Python-based trivia quiz game that tests your knowledge with true/false questions across various topics.

## Description
This interactive command-line quiz game presents users with a series of true/false questions, keeps track of their score, and provides immediate feedback after each answer. The game is designed to be educational and entertaining for users of all ages.

## How to Play
Run the Python script

Read each question carefully

Type True or False (case-insensitive) to answer

Receive immediate feedback on your answer

View your updated score after each question

Complete all questions to see your final score

### Game Logic
The program:

Presents questions one at a time

Validates user input (True/False answers)

Compares user answers with correct answers

Updates and displays the score

Provides educational feedback with correct answers
