Usage
Starting the Program
Run the coffee machine program to begin serving customers.

Ordering a Drink
When prompted, enter your drink choice: espresso, latte, or cappuccino

The machine will check if sufficient resources are available

If resources are sufficient, you'll be prompted to insert coins

Making Payment
Enter the number of each coin type when prompted:

How many quarters?

How many dimes?

How many nickles?

How many pennies?

Transaction Outcomes
Successful: Drink is dispensed with enjoyment message

Insufficient Funds: Money refunded with apology

Excess Payment: Change provided along with drink

Maintenance Commands
report: Displays current resource levels and profit

off: Securely shuts down the coffee machine
